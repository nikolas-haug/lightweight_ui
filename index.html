About

Pineapple Frontend is a lightweight CSS/SASS framework with simple components and a fully customizable flexbox grid system. It also includes several lightweight Javascript componenets that are compatible with a Wordpress custom theme.

I created it in order to better understand dynamic flexbox grids using SCSS loops and mixins, particularly following the way the Bootstrap framework generates these grids with SCSS variables and functions. I wanted to make something a little more lightweight that was simple to customize for projects of all size.

Work Flow

Download the framework folder and then run npm i from the command line. After modules are installed, in the root of your project, run

npm run start

to start the local live server. node-sass will watch for changes to the source files.

Using the npm scripts with node-sass and plugins, the Sass files are prefixed and compiled in development and production environments.

"scripts": {
    "devserver": "live-server",
    "start": "npm-run-all --parallel devserver dev",
    "dev": "node-sass ./main.scss ./dist/style.css -w",
    "build": "node-sass ./main.scss ./dist/style.css --output-style compressed",
    "prefix": "postcss --use autoprefixer -b 'last 5 versions' dist/style.css -o dist/style.prefix.css",
    "compress": "node-sass dist/style.prefix.css dist/style.min.css --output-style compressed",
    "build:all": "npm-run-all build prefix compress"
},

Define global styles for the project in the variables.scss file . The base font size adjusts the root default, and all widths, heights, font-sizes, paddings, and margins are set using rems, where 1rem is equal to 10px.

//=====================================================
// TYPOGRAPHY
//=====================================================

// Set the default to 1rem size for browser window resize
$base-font-size: 62.5%;
$base-line-height: 1.6;

$font-size-primary: 1.6rem;

Containers

There are four container sizes. The container variables set the four default breakpoint widths. 

$x-small: 480px;
$small: 750px;
$medium: 992px;
$large: 1200px;

// Sets the width of the containers
$container-gutter: 0.95;

In mixins.scss, the container gutters are calcualted using the defined widths and the container gutter variable.

@mixin container-size($size, $gutter) {
    max-width: calc(#{$size} * #{$gutter}); 
}
                                
// MAIN CONTAINER BREAKPOINTS 

@include x-small-breakpoint {
    .container {
    @include container-size($x-small, $container-gutter);
    }
}

@include small-breakpoint {
    .container {
    @include container-size($small, $container-gutter);
    }
}

@include medium-breakpoint {
    .container {
    @include container-size($medium, $container-gutter);
    }
}

@include large-breakpoint {
    .container {
    @include container-size($large, $container-gutter);
    }
}

Layout

The grid layout is built using Flexbox and is fully customizable. Define the number of grid columns and column padding in variables.scss.

//=====================================================
// GRID
//=====================================================

// Controls the number of columns for the layout
$grid-columns: 12;
$grid-column-padding: 1.5rem;

In layout.scss the columns are generated in for loops using the defined grid variable.

@for $i from 1 through $grid-columns {
    .col-#{$i} {
        flex-basis: percentage($i / $grid-columns);
        max-width: percentage($i / $grid-columns);
    }
    .col-offset-#{$i} {
        margin-left: percentage($i / $grid-columns);
    }
}

The defined padding variable is added to all generated columns.
