About

Pineapple Frontend is a lightweight CSS/SASS framework with simple components and a fully customizable flexbox grid system. It also includes several lightweight Javascript componenets that are fully compatible with Wordpress custom themes.

Inspiration and credit for this design goes to:

  • Tania Rascia: Primitive UI
  • Kristofer Joseph: Flexbox Grid

Work Flow

Download the framework folder and then run npm i from the command line. After modules are installed, in the root of your project, run npm run start to start the local live server. node-sass will watch for changes to the source files.

Using the npm scripts with node-sass the Sass files are prefixed and compiled.

"scripts": {
    "devserver": "live-server",
    "start": "npm-run-all --parallel devserver dev",
    "dev": "node-sass ./main.scss ./dist/style.css -w",
    "build": "node-sass ./main.scss ./dist/style.css --output-style compressed",
    "prefix": "postcss --use autoprefixer -b 'last 5 versions' dist/style.css -o dist/style.prefix.css",
    "compress": "node-sass dist/style.prefix.css dist/style.min.css --output-style compressed",
    "build:all": "npm-run-all build prefix compress"
},

The variables.scss file is where to define the global styles for the project.

Containers

There are four container sizes. The container variables set the four default breakpoint widths. 

$x-small: 480px;
$small: 750px;
$medium: 992px;
$large: 1200px;

// Sets the width of the containers
$container-gutter: 0.95;

Using mixins, the container gutters are calcualted using the defined widths and the container gutter variable. 

@mixin container-size($size, $gutter) {
    max-width: calc(#{$size} * #{$gutter}); 
}
                                
// MAIN CONTAINER BREAKPOINTS 

@include x-small-breakpoint {
    .container {
    @include container-size($x-small, $container-gutter);
    }
}

@include small-breakpoint {
    .container {
    @include container-size($small, $container-gutter);
    }
}

@include medium-breakpoint {
    .container {
    @include container-size($medium, $container-gutter);
    }
}

@include large-breakpoint {
    .container {
    @include container-size($large, $container-gutter);
    }
}
